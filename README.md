## Calculator

**Basic Features**

- Basic Arithmetic Operations – Supports addition (+), subtraction (-), multiplication (*), and division (/).
- Numeric Keypad – Buttons 0–9 for entering numbers.
- Clear Function (C) – Clears the current input.
- Equals Button (=) – Calculates and shows the result.
- Simple Grid Layout – Organized 4x4 button grid for easy usability.
- Touchscreen-Friendly Buttons – Large buttons suitable for mobile devices.
- Minimalist Design – Clean interface with focus on functionality.
- Decimal Point (.) – Supports non-integer and precise decimal calculations.
- Percentage (%) Function – Easily compute percentages for quick calculations.
- History with Display Scrolling – View and scroll through previous calculations without losing your current result.
